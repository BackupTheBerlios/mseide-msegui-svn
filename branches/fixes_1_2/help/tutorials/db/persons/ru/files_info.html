Файлы исходников полностью рабочего проекта
( на момент написания данного руководства )

Скачайте файл persons.tgz.

Распакуйте это архив в каталог, где программист ( Вы ? ) имеет доступ на запись, и сделайте этого програмиста владельцем всех этих файлов, естетственно - тоже с правами на запись.
Не забудьте, что такие же владелец и права доступа должны быть и на файлы и каталоги самой MSEgui-библиотеки.

Запуск проекта :

  • закройте предыдущий проект и, через "Project -> Open", укажите на файл "persons.prj" в вышеозначенном каталоге.
    • этот файл помнит последнее местонахождение файлов проекта ( на файловой системе моего рабочего места ), поэтому, в некоторых редких случаях - может потребовать от Вас показать, где какие файлы оказались.
  • далее, как обычно :
    • Project -> Build : пересобрать все ( даже неизмененные ) исходные файлы, с опциями по умолчанию
    • Project -> Make ( или "F9" ) : пересобрать только измененные исходные файлы, и запустить вновь собранный исполняемый файл ( с поддержкой пошаговой отладки - то бишь с остановками на "breakpoints" )

Если что-то пошло совсем не так, то удалите файлы "*_mfm.pas " и переоткройте проект заново - каждый из этих файлов будет пересоздан в момент сборки проекта после малейшего изменения ( потаскать туда-сюда и .т.п. ) в состветствующей форме ( *.mfm -файле ) . MSEide в эти моменты может Вас попросить указать на нужный *.mfm -файл ( будет видно, какой ) .
